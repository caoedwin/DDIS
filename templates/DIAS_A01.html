📜 角色 Rule 說明
系統採 5 階角色制（Admin / TL / QM / TE / RD）。Admin 為最高權限；TL 負責審核與專案內人員管理；QM（Quality Manager）權限同 TL；TE 負責建立 issue；RD 負責 issue 分類與處理。
Admin SYSTEM ADMIN
✓ 專案管理：建立 / 編輯 / 刪除
✓ 使用者管理：新增 / 刪除 / 權限分配 / issue 狀態修改
✓ PAW Group 管理：建立 / 刪除
✓ 所有 Issue：查看 / 審核 / 狀態修改 / 刪除 / 匯出 Excel
